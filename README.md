# 系统天气应用竞品分析技能

这是一个专门用于系统天气应用竞品分析的技能文档。

## 文件说明

- **SKILL.md**: 系统天气应用竞品分析完整技能文档

## 技能概述

此技能用于对系统天气应用进行全面的竞品分析与对标，输出专业的 Markdown 报告。

### 覆盖范围

- iOS 天气（Apple Weather）
- 小米天气
- 鸿蒙纯血版天气（HarmonyOS NEXT）
- 荣耀天气
- 墨迹天气

### 核心能力

1. **功能矩阵对比**: 统一维度的功能点对比分析
2. **体验走查**: 详细的用户体验走查与证据点收集
3. **差异洞察**: 识别关键差异与产品机会点
4. **优先级建议**: P0/P1/P2 级别的改进建议与落地方案

### 分析维度

- UX/信息架构
- 预报能力（逐小时/逐日/分钟级）
- 生活指数
- 系统能力（小组件/通知/卡片）
- 商业化策略
- 数据来源与质量
- AI 能力

### 使用场景

当用户提到以下关键词时使用此技能：
- 竞品分析 / 对标
- 天气 app
- iOS 天气 / 小米天气 / 鸿蒙天气 / 荣耀天气 / 墨迹天气
- 系统天气

## 输出标准

- **格式**: 结构化 Markdown 报告
- **风格**: 产品经理式，结论先行
- **质量**: 可复查的证据点 + 入口路径说明
- **深度**: 包含对比表、证据点、结论与优先级建议
